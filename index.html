Complete CSS Properties List

1. Box Model

The Box Model diagram illustrates content, padding, border, and margin.

  • width – Sets the width of an element.
    More details

    The width property determines the horizontal space allocated for an element's content. It can accept various units such as pixels (px), percentages (%), ems (em), rems (rem), etc. It does not include padding, border, or margin unless box-sizing: border-box; is set.

  • height – Sets the height of an element.
    More details

    Similar to width, the height property sets the vertical space for an element. It can also take multiple unit types. Note that content may overflow if the content’s size exceeds the specified height.

  • margin – Sets outer spacing around an element.
    More details

    The margin property creates space outside an element's border. It accepts one to four values, which define the space on the top, right, bottom, and left sides. Margins can collapse in vertical directions.

  • padding – Sets inner spacing inside an element.
    More details

    Padding is the space between the element’s content and its border. Like margins, it accepts up to four values corresponding to each side. Padding adds to the overall size of an element unless using box-sizing: border-box;.

  • border – Defines the border style, width, and color.
    More details

    The border shorthand property allows you to set the width, style, and color of an element's border. It is equivalent to specifying border-width, border-style, and border-color separately.

  • box-shadow – Adds a shadow to an element.
    More details

    box-shadow creates a shadow effect around an element's frame. You can control the horizontal and vertical offsets, blur radius, spread radius, and color. This property can add depth or focus to elements.

2. Backgrounds & Colors

Visual example of background properties applied to an element.

  • background-color – Defines the background color.
    More details

    This property sets a solid color as the background for an element. It accepts color names, hex codes, RGB, RGBA, HSL, and HSLA values. It is often used with transparency (via RGBA or HSLA) to create overlay effects.

  • background-image – Adds a background image.
    More details

    The background-image property allows you to set an image as the background of an element. You can use URL values to specify the location of the image. It is often combined with other background properties to control repetition and positioning.

  • background-size – Sets the size of the background image.
    More details

    background-size determines how the background image is scaled. Common values include cover (covering the entire element), contain (scaling the image to fit inside the element), or specific dimensions (like 100px 50px).

  • opacity – Sets the transparency of an element.
    More details

    The opacity property makes an element and its contents partially or fully transparent. Values range from 0 (completely transparent) to 1 (fully opaque). It is useful for creating fade effects.

3. Typography & Text Styling

Example of various text styling properties in action.

  • color – Sets the text color.
    More details

    The color property specifies the color of the text content. It accepts color names, hex codes, and RGB/RGBA values. It is fundamental for ensuring text visibility and meeting accessibility standards.

  • font-size – Defines the font size.
    More details

    This property sets the size of the text. You can use absolute units like pixels (px) or relative units like ems (em) and rems (rem). Responsive typography often uses relative units to adapt to different devices.

  • text-align – Aligns text (left, center, right).
    More details

    text-align specifies the horizontal alignment of text within its container. Common values are left, center, right, and justify. It helps in creating well-structured and visually appealing text blocks.

  • text-decoration – Adds underline, overline, etc.
    More details

    This property adds visual decorations to text, such as underlines, overlines, and line-throughs. It is often used to indicate links or to show that text has been removed or modified.

4. Display & Visibility

Diagram showing how display and visibility affect layout.

  • display – Defines element layout (block, flex, inline, etc.).
    More details

    The display property determines how an element is rendered in the document flow. It can take values such as block, inline, flex, grid, and more. This property is essential for controlling layout behavior.

  • visibility – Shows or hides an element.
    More details

    visibility controls whether an element is visible or hidden, while still taking up space in the layout. Common values are visible and hidden.

  • overflow – Defines content overflow behavior.
    More details

    This property controls what happens when an element’s content exceeds its allocated space. Values include visible, hidden, scroll, and auto. It is especially useful when dealing with fixed-size containers.

  • z-index – Controls element stacking order.
    More details

    The z-index property specifies the stacking order of positioned elements (those with a position other than static). Higher values are placed on top of lower values.

5. Positioning & Layout

Visual representation of positioning properties: static, relative, absolute, fixed, and sticky.

  • position – Defines how an element is positioned.
    More details

    The position property allows you to specify how an element is placed in the document. Its values include static (default), relative, absolute, fixed, and sticky.

  • top, right, bottom, left – Moves an element in different directions.
    More details

    These properties work with positioned elements (non-static) to define offsets from the respective edges of their containing block. They enable precise control over element placement.

  • float – Aligns elements left or right.
    More details

    The float property removes an element from the normal document flow and aligns it to the left or right of its container. It is commonly used for wrapping text around images, although modern layouts often use flexbox or grid instead.

  • clear – Controls floating elements.
    More details

    The clear property specifies on which sides of an element floating elements are not allowed. It helps to prevent layout issues when using floats.

6. Flexbox

Diagram illustrating how flexbox distributes space among items.

  • display: flex – Enables flexbox layout.
    More details

    Setting an element’s display to flex initiates a flex container, allowing for dynamic alignment and distribution of space among child items. Flexbox simplifies vertical and horizontal centering.

  • justify-content – Aligns items horizontally.
    More details

    This property aligns flex items along the main axis (usually horizontal). Values such as flex-start, flex-end, center, space-between, and space-around determine the spacing between items.

  • align-items – Aligns items vertically.
    More details

    Within a flex container, align-items aligns items along the cross axis (usually vertical). It accepts values like flex-start, center, flex-end, and stretch.

  • flex-wrap – Controls item wrapping.
    More details

    The flex-wrap property allows flex items to wrap onto multiple lines when there isn’t enough space in one line. It can take the values nowrap (default), wrap, or wrap-reverse.

7. Grid Layout

Grid layout diagram showing columns, rows, and gaps.

  • display: grid – Enables CSS Grid layout.
    More details

    When an element’s display is set to grid, it becomes a grid container for laying out child elements in rows and columns. CSS Grid offers two-dimensional control, unlike flexbox which is mostly one-dimensional.

  • grid-template-columns – Defines column structure.
    More details

    This property specifies the number, widths, and patterns of columns in a grid layout. Values can be fixed lengths, percentages, fractions (fr), or a mix.

  • grid-template-rows – Defines row structure.
    More details

    Similar to grid-template-columns, this property sets the number, heights, and patterns of rows in a grid. It allows for dynamic and responsive grid designs.

8. Transitions & Animations

Visual example showing smooth transitions and animations.

  • transition – Adds smooth transitions.
    More details

    The transition property allows you to define the transition effect for changes in CSS property values. You can set the duration, timing function, and delay. It is commonly used to animate hover effects or state changes.

  • animation – Applies animations.
    More details

    The animation property is a shorthand for setting multiple animation properties, including the animation name, duration, timing function, delay, iteration count, and direction. It is used for more complex, keyframe-based animations.

  • animation-duration – Sets how long an animation runs.
    More details

    This property defines the length of time an animation takes to complete one cycle. It accepts time values like seconds (s) or milliseconds (ms).

9. Transformations

Demonstration of transformation effects like rotate, scale, and translate.

  • transform – Applies transformations.
    More details

    The transform property allows you to modify the coordinate space of an element. It supports operations like rotate, scale, translate, and skew. Multiple transform functions can be applied at once.

  • rotate – Rotates an element.
    More details

    Using rotate() within the transform property, you can turn an element by a specified angle (in degrees, radians, etc.). It is useful for creating dynamic and interactive UI effects.

  • scale – Resizes an element.
    More details

    The scale() function in the transform property adjusts the size of an element. Values greater than 1 enlarge it, and values less than 1 shrink it. Scaling can be applied uniformly or separately on the X and Y axes.

10. Filters & Effects

Visual examples of filter effects such as blur, brightness, and contrast.

  • filter – Applies visual effects.
    More details

    The filter property applies graphical effects like blur, brightness, contrast, grayscale, hue rotation, invert, opacity, saturate, and sepia to an element. These effects are rendered in the browser, enhancing visual appeal without needing to edit images.

  • backdrop-filter – Applies effects to background content.
    More details

    Unlike filter, the backdrop-filter applies effects to the area behind an element, such as blurring or color shifting the background. This is commonly used for frosted-glass effects in overlays and modal dialogs.

11. Cursor & User Interaction

Example showing different cursor styles when hovering over elements.

  • cursor – Defines the cursor type.
    More details

    The cursor property specifies the mouse cursor that appears when hovering over an element. Common values include pointer (hand icon), default, crosshair, and move. It improves user interactivity by providing visual cues.

  • pointer-events – Controls clickability.
    More details

    This property determines under what circumstances (if any) an element can be the target of mouse events. For example, setting it to none will disable clicks, hover effects, and other pointer events on the element.

12. Media Queries (Responsive Design)

Diagram explaining how media queries adjust layouts for different screen sizes.

  • @media – Applies styles based on screen size.
    More details

    Media queries enable you to apply CSS styles conditionally based on characteristics such as screen width, height, orientation, and resolution. They are fundamental for responsive web design.

  • max-width – Targets smaller screens.
    More details

    In media queries, max-width specifies an upper limit for the viewport width, allowing you to apply styles when the screen is smaller than the defined value.

  • min-width – Targets larger screens.
    More details

    Conversely, min-width sets a lower limit on the viewport width, applying styles when the screen is at least the specified size. This is crucial for creating mobile-first designs.

13. Miscellaneous Properties

Overview of additional properties that enhance functionality.

  • content – Inserts content.
    More details

    The content property is used with pseudo-elements (::before and ::after) to insert generated content. It can be text, images (via URL), or even counters, and is essential for decorative and functional purposes.

  • resize – Allows resizing of elements.
    More details

    This property specifies whether an element is resizable by the user. Common values include none, both, horizontal, and vertical. It is often used with textareas and other input elements.

  • scroll-behavior – Controls scrolling.
    More details

    The scroll-behavior property enables smooth scrolling when navigating to an anchor within the page or programmatically scrolling. It can be set to auto (default) or smooth.
